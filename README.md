WhatsApp Bulk Message Sender with Analytics Dashboard
Overview

This project is an advanced version of a traditional WhatsApp Bulk Message Sender.
Along with automated WhatsApp message delivery, this system also includes:

📊 Real-time Analytics Dashboard
🗄️ MySQL Database Integration
🌐 Interactive Frontend using HTML, CSS & JavaScript
⚡ Flask Backend APIs
🔄 Real-time Updates with Socket.IO
📁 CSV-based Contact Management

The project automates personalized WhatsApp messaging while tracking message statistics such as:

Total Messages Sent
Delivered Messages
Read Messages
Failed Messages
Features
✅ Bulk WhatsApp Messaging
Send personalized messages to multiple contacts
CSV-based contact importing
Dynamic message templates using placeholders
✅ Analytics Dashboard

✅ Database Connectivity
